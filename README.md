# Sistema de Control de Equipos de Apilamiento

Plataforma empresarial para la gestión operativa de equipos de apilamiento alquilados, orientada al control de trazabilidad, averías, evidencias fotográficas, tiempos de inactividad e indicadores operativos.

---

# Componentes del Proyecto

| Componente | Descripción |
|---|---|
| Mobile App | Aplicativo Android operativo |
| Backend API | Servicios REST y lógica de negocio |
| Dashboard Web | Indicadores y gestión administrativa |
| Database | Persistencia de información |
| Documentation | SDD y documentación técnica |
